# MNIST Handwritten Digit Classification

## Project Overview
This project compares multiple machine learning models for handwritten digit recognition using the MNIST dataset.

## Models Used
- KNN
- SVM
- Logistic Regression
- Naive Bayes
- Decision Tree

## Results
Best model: SVM  
Accuracy on MNIST: 97.92%  
Accuracy on custom handwritten digits: 80%

## Dataset
MNIST PNG Dataset:
https://www.kaggle.com/datasets/hojjatk/mnist-dataset

## Features
- Data preprocessing
- Model comparison
- Confusion matrix analysis
- Custom handwritten digit testing

## Tools
Python, OpenCV, NumPy, Scikit-learn, Matplotlib
